## Install

With npm installed run the following commands on project folder:

```bash
  #Download Libraries
  npm install
  #Installing Angular CLI:
  npm install -g @angular/cli
  #Start the project using Angular CLI:
  ng serve
```
    
[Angular CLI documentation](https://angular.io/docs)
